MERN Portfolio Project

About

This is a personal portfolio web application built using the MERN Stack.

Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- HTML
- CSS
- JavaScript

Features

- Responsive Portfolio Website
- About Section
- Skills Section
- Projects Section
- Contact Form
- MongoDB Integration
- REST API Backend
- GitHub Profile Link
- LinkedIn Profile Link

Folder Structure

portfolio_project/

├── client/

├── server/

└── README.md

Installation

Frontend

cd client

npm install

npm start

Backend

cd server

npm install

npm start
