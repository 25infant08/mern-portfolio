#**MERN PORTFOLIO WEBSITE**

---

Introduction

This repository contains a MERN Stack Portfolio Website developed to showcase personal information, technical skills, projects, and contact details through a modern and responsive web application. The project demonstrates full-stack web development concepts including frontend development, backend API creation, and database integration.

---

Project Overview

Overview

A full-stack portfolio application built using React.js, Node.js, Express.js, and MongoDB.

Features

* Responsive Portfolio Website
* About Section
* Skills Section
* Projects Section
* Contact Form
* REST API Integration
* MongoDB Database Connectivity
* GitHub Profile Link
* LinkedIn Profile Link

Technologies Used

Frontend

* React.js
* HTML
* CSS
* JavaScript

Backend

* Node.js
* Express.js

Database

* MongoDB

---

How to Run

Frontend

cd client
npm install
npm start

Backend

cd server
npm install
npm start

Learning Outcomes

* Full Stack Web Development
* React Component Development
* REST API Development
* Express.js Backend Development
* MongoDB Integration
* CRUD Operations
* Git and GitHub Version Control
